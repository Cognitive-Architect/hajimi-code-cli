## 1. 问题陈述

### 1.1 原实现问题

**`src/test/lsh-collision-sim.js` (v1 简化版)**:

```javascript
// 第35-61行：简化版 SimHash 实现
function simhash64(vector) {
  // 简化的 SimHash: 使用随机投影
  const bits = new Array(64).fill(0);
  
  for (let i = 0; i < 64; i++) {
    let projection = 0;
    for (let j = 0; j < vector.length; j++) {
      // 使用MD5派生权重（问题所在）
      const hash = crypto.createHash('md5')
        .update(`projection_${i}_${j}`)
        .digest();
      const weight = (hash.readInt32LE(0) % 2000) / 1000 - 1;
      projection += vector[j] * weight;
    }
    bits[i] = projection >= 0 ? 1 : 0;
  }
  // ...
}
```

**问题**:
1. 使用 MD5 派生权重，不符合 LSH 理论要求的随机超平面投影
2. 权重分布不均匀，可能导致汉明距离分布偏离理论预期
3. 与生产环境实现不一致，验证结果不可迁移

### 1.2 修复方案

| 项目 | 原实现 (v1) | 修复后 (v2) |
|------|-------------|-------------|
| 投影向量生成 | MD5派生 | Box-Muller正态分布 |
| 汉明距离分布 | 可能偏离32 | 峰值严格在32附近 |
| 理论符合度 | 低 | 高（符合LSH理论） |
| 生产一致性 | 不一致 | 一致 |
