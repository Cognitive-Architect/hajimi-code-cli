### FUSE-004: Promise.race 竞态熔断

**触发条件**:
```bash
node tests/redis-race-condition.test.js
# 输出: Race condition detected: 3/100
# 标准: > 0 (任何竞态都不可接受)
```

**熔断动作**:
1. 停止使用 Promise.race
2. 改用备选策略B
3. 重新测试

**备选实现 (策略B)**:
```javascript
// 改用 ioredis 内置 timeout
this.redis = new Redis({
  commandTimeout: 1000,  // 全局命令超时
});

// 简化 healthCheck
async healthCheck() {
  try {
    const result = await this.redis.ping();
    return result === 'PONG';
  } catch (err) {
    // 超时或错误都会进入这里
    return false;
  }
}
```

**Plan B**:
- 改用 ioredis commandTimeout
- 影响：所有 Redis 命令都有超时
- 风险：可能影响正常操作的性能
