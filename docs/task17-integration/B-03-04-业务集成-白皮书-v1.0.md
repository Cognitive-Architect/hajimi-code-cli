## 第2章：实现细节

### 2.1 中间件挂载

```javascript
app.use(rateLimitMiddleware.middleware());
```

### 2.2 熔断器状态机

```
CLOSED → failures>threshold → OPEN → timeout → HALF_OPEN → success → CLOSED
```

### 2.3 降级策略

- SQLite故障 → 内存降级
- 限流器错误 → 放行

## 第3章：熔断策略

| 状态 | 行为 |
|:---|:---|
| CLOSED | 正常限流 |
| OPEN | 拒绝所有请求 |
| HALF_OPEN | 允许试探请求 |
