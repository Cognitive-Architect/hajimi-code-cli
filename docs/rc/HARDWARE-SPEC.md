# RC性能验证硬件规格

## 实际硬件
- **CPU**: AMD Ryzen 7 8845H (8核, 3.8GHz)
- **RAM**: 32GB (标注: 超过消费级<16GB约束，需模拟限制)
- **磁盘**: 300GB SSD
- **系统**: Windows 11

## 消费级约束适配
- **测试规模**: 100MB（非10GB，适配消费级存储）
- **内存熔断**: 300MB（约束RSS，模拟消费级内存压力）
- **CPU温度**: <85°C警告，90°C熔断
- **串行执行**: 禁止Promise.all，强制串行任务执行

## 差异声明
> ⚠️ **重要**: 实际RAM(32GB) > 消费级约束(<16GB)，但通过300MB熔断线模拟约束环境。
> 
> 本测试在硬件充足的环境中运行，但通过软件熔断机制强制模拟消费级设备的资源限制，
> 确保代码在受限环境下的可靠性。

## 测试参数基准
| 参数 | 消费级约束 | 实际硬件 | 模拟方式 |
|------|-----------|----------|----------|
| 内存上限 | <16GB | 32GB | 300MB熔断线 |
| 测试数据 | 10MB-100MB | 100MB | 固定100MB |
| 采样间隔 | 30秒 | 30秒 | 精确匹配 |
| 执行模式 | 串行 | 串行 | 框架强制 |

## 监控指标
- RSS内存: 通过 `process.memoryUsage().rss` 监控
- 熔断阈值: 314,572,800 bytes (300MB)
- 日志格式: CSV (`timestamp,rss,heapUsed,heapTotal`)
