## 🔧 9大功能系统

```
┌─────────────────────────────────────────────────────────┐
│ 1. 核心底座 (Core Kernel) - 永不裁剪                     │
│    QueryEngine + ToolRegistry + EventLoop               │
├─────────────────────────────────────────────────────────┤
│ 2. 工具系统 (40+ Tools) - 可开关                         │

├─────────────────────────────────────────────────────────┤
│ 3. UI系统 (3 Modes) - 可切换                             │
│    终端(Ink) / Web(React) / VS Code插件                  │
├─────────────────────────────────────────────────────────┤
│ 4. 记忆系统 (5 Layers) - 可配置                          │
│    Session → Auto → Dream → Graph → Cloud               │
├─────────────────────────────────────────────────────────┤
│ 5. 上下文压缩 (3 Levels) - 自动触发                      │
│    micro → auto → compact (+ Cascade可选)               │
├─────────────────────────────────────────────────────────┤
│ 6. LLM插件 (3+ Providers) - 可切换                       │
│    Claude / OpenAI / Ollama本地                          │
├─────────────────────────────────────────────────────────┤
│ 7. 同步系统 (3 Ways) - 可选开启                          │
│    P2P / Git / Cloud                                     │
├─────────────────────────────────────────────────────────┤
│ 8. 安全系统 (Multi-layer) - 可配置                       │
│    Sandbox + Permission + Audit                          │
├─────────────────────────────────────────────────────────┤
│ 9. 个人知识库 (Ultimate) - 核心差异化                    │
│    代码索引 + 风格学习 + 习惯分析 + 知识图谱              │
└─────────────────────────────────────────────────────────┘
```

## ✅ 功能开关速查

```json
["ui:terminal", "tools:file", "tools:bash", "memory:session", "llm:anthropic"]
```

```json
["ui:terminal", "tools:file", "tools:bash", "tools:git", "tools:search", 
 "tools:build", "memory:session", "memory:auto", "llm:anthropic"]
```

```json
["ui:all", "tools:all", "memory:all", "llm:all", "compression:all", 
 "sync:all", "security:all", "personalization:all"]
```

## 🎨 设计理念

1. **模块化** - 每个功能独立开关
2. **可积累** - 越用越懂你的个人知识库
3. **本地优先** - 隐私 + 速度
4. **深度定制** - 不是给所有人用，是给你用
