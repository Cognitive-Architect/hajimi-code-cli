### 关键技术验证

**Archive格式（代码实现）**：
```rust
// L11: 魔数定义
const HCTX_MAGIC: &[u8; 4] = b"HCTX";

// L25-31: 写入顺序
file.write_all(HCTX_MAGIC)?;  // magic(4)
file.write_all(&[HCTX_VERSION, 0, 0, 0])?;  // version(1)+flags(1)+reserved(2)
file.write_all(&(json_bytes.len() as u32).to_le_bytes())?;  // body_len(4)
file.write_all(&json_bytes)?;  // body(N)
file.write_all(blake3::hash(&json_bytes).as_bytes())?;  // BLAKE3(32) ✅ AFTER body
```
