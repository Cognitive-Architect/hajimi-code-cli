## ✅ 本周落地清单 (4项核心任务)

### 1. 修复 uint64 → Number 精度坑 (RISK-H-001)
**状态**: ✅ 已完成

```typescript
// ✅ 正确: 全程 BigInt
const simhash = buf.readBigUInt64LE(0);  // bigint

// ❌ 错误: 会截断

```

### 3. 候选检索实现 + 压测门禁 (RISK-H-003)
**状态**: ⏳ 待执行

**问题**: `popcnt(a ^ b) < 3` 在 N=1e8 (TB级) 时，邻居查找 2081 次/块，总计 2e11 次 lookup。

**推荐方案**:
```
1. 多索引 LSH: 将 64 位切成 4 段 16 位，建立倒排索引
2. 滑窗优化: 只在最近 K 个 chunk 内搜索 (K 可配置)
3. 压测门禁: 10M chunk 压测，确保吞吐不跌穿预算
```

**验收标准**:
- [ ] 10M chunk 压测通过
- [ ] 吞吐不低于基线 80%

---
