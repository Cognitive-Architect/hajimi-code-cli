# Task 工单文件夹

> 用户发放工单的地方

## 使用说明

1. **用户**把任务内容写入 `task/` 下的 markdown 文件
2. **我**读取该文件，理解任务要求
3. **聊天框**里只需说："执行 task/xxx.md"

---

## 任务文件格式

```markdown
# 任务标题

## 任务描述
简要说明要做什么

## 输入
- 输入文件路径
- 或者需要处理的内容

## 输出要求
- 输出文件路径
- 格式要求

## 约束条件
- 性能要求
- 其他限制

## 验收标准
- [ ] 标准1
- [ ] 标准2
```

## 如何发布新任务

1. 在 `task/` 下创建新文件，如 `task-001.md`
2. 按上方格式填写任务内容
3. 在聊天框说："执行 task/task-001.md"
4. 我会读取并执行

---

> 💡 提示：复杂任务建议拆分成多个小任务文件
