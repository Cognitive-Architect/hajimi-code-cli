# 示例任务：批量重命名文件

## 任务描述
将 workspace/assets/ 下的所有图片文件按日期重命名

## 输入
- 路径: `workspace/assets/`
- 文件类型: `.jpg`, `.png`

## 输出要求
- 输出路径: 原目录
- 命名格式: `IMG_YYYYMMDD_序号.扩展名`
- 示例: `IMG_20240222_001.jpg`

## 约束条件
- 不覆盖已有文件
- 保持原始文件创建时间
- 生成操作日志

## 验收标准
- [ ] 所有图片文件被重命名
- [ ] 命名格式正确
- [ ] 生成重命名日志文件
- [ ] 无文件丢失
